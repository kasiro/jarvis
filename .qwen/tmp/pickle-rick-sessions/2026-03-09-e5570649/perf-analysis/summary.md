## 🎯 ОДНА ГЛАВНАЯ ПРОБЛЕМА

## ✅ БЫСТРОЕ РЕШЕНИЕ (5 минут)

### Изменить silence_threshold с 0.8 на 0.4 сек

**Файл:** `crates/jarvis-app/src/app.rs`, строка 35

```rust
// БЫЛО:
let silence_threshold: u32 = ((0.8 * sample_rate as f32) / frame_length as f32) as u32;

// СТАЛО:
let silence_threshold: u32 = ((0.4 * sample_rate as f32) / frame_length as f32) as u32;
```

**Результат:**
- ⚡ Задержка: **-400 мс** (с 800 до 400 мс)
- ⚡ Ассистент ощущается в **2 раза быстрее**
- ⚠️ Риск: возможны преждевременные обрезания (тестировать!)

## 🛡️ Как избежать ложных срабатываний

### 1. Увеличить VAD порог (если много ложных)
```rust
// config.rs:183
pub const VAD_ENERGY_THRESHOLD: f32 = 200.0;  // Было 100.0
```

### 2. Включить ML-based VAD (лучшее качество)
```rust
// config.rs:77
pub const DEFAULT_VAD_BACKEND: &str = "nnnoiseless";
```

### 3. Добавить минимальную длительность речи
```rust
// app.rs: добавить после vad_state
let min_voice_frames: u32 = 2;  // 64 мс минимум
```
