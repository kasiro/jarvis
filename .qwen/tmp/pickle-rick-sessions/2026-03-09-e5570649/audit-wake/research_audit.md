## 📝 Рекомендации по исправлению

### 🔴 HIGH Priority - Критические команды

#### 1. Добавить `wake_word_required: false` для `reboot`

**Файл:** `/home/kasiro/Документы/jarvis/resources/commands/jarvis/command.yaml`

**Текущее состояние:**
```yaml
- command:
    action: python
    script: reboot.py
  voice:
    sounds:
      - ok1
      - ok2
      - ok3
  phrases:
    - перезагрузись
    - перезагрузи себя
    - выполни перезагрузку
```

**Исправление:**
```yaml
- command:
    action: python
    script: reboot.py
    wake_word_required: false  # ← ДОБАВИТЬ
  voice:
    sounds:
      - ok1
      - ok2
      - ok3
  phrases:
    - перезагрузись
    - перезагрузи себя
    - выполни перезагрузку
```

**Обоснование:** Критическая команда управления Jarvis, должна работать без wake word.
