# Research: Mode Switch Commands

**Date**: 2026-02-26

## 1. Executive Summary

Created Lua-based voice commands for mode switching, replacing Python scripts with Lua API integration.

## 2. Technical Context

### Existing Implementation
- **Location**: `resources/commands/modes/`
- **Format**: Python scripts (`kid_mode_on.py`, `kid_mode_off.py`)
- **Issue**: No integration with Lua API, direct Python imports

### New Implementation
- **Format**: Lua scripts with `jarvis.modes` API
- **Commands**: kid_mode_on, kid_mode_off, dev_mode_on, check_mode
- **Configuration**: `command.yaml` updated

## 3. Findings & Analysis

### Command Structure

#### kid_mode_on.lua
- Switches to "kid" mode
- Shows notification
- Returns success/failure

#### kid_mode_off.lua
- Switches to "normal" mode
- Shows notification
- Returns success/failure

#### dev_mode_on.lua
- Switches to "dev" mode
- Shows notification
- Uses "full" sandbox for dev features

#### check_mode.lua
- Shows current mode
- Lists available modes
- Uses "minimal" sandbox

### Configuration (command.yaml)

```yaml
list:
  - command:
      type: lua
      script: kid_mode_on.lua
      sandbox: standard
    voice:
      sounds: [ok1, ok2, ok3]
    phrases:
      ru: ["детский режим", "запусти детский режим", ...]
      en: ["kid mode", "enable kid mode", ...]
```

## 4. Technical Constraints

- Lua scripts require `jarvis.modes` API (ticket f350fce3)
- Sandbox levels: minimal, standard, full
- Phrases must be in Russian and English

## 5. Architecture Documentation

### File Structure
```
resources/commands/modes/
├── command.yaml           # Command configuration
├── kid_mode_on.lua        # NEW: Kid mode activation
├── kid_mode_off.lua       # NEW: Kid mode deactivation
├── dev_mode_on.lua        # NEW: Dev mode activation
├── check_mode.lua         # NEW: Check current mode
├── kid_mode_on.py         # Legacy Python (kept for compatibility)
└── kid_mode_off.py        # Legacy Python (kept for compatibility)
```

### Usage Examples

**Voice Command (Russian):**
```
"запусти детский режим"
→ kid_mode_on.lua
→ jarvis.modes.set_mode("kid")
→ Notification: "Детский режим активирован"
```

**Voice Command (English):**
```
"disable kid mode"
→ kid_mode_off.lua
→ jarvis.modes.set_mode("normal")
→ Notification: "Деактивирован детский режим"
```
