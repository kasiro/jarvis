### 0. 🔍 ПРОВЕРКА ВСЕХ КОМАНД (ПЕРВАЯ ЗАДАЧА!)

**Статус:** 🔴 Критично

**Описание:**
Прежде чем добавлять новые функции, нужно **убедиться что все существующие команды работают корректно**!

---

### Шаг 1: Проверить `resources/commands/ADD.md`

**Что делать:**
1. Открыть `resources/commands/ADD.md`
2. Проверить таблицу команд
3. Отметить какие команды требуют проверки

---

### Шаг 2: Проверить `resources/commands/FIX.md`

**Что делать:**
1. Открыть `resources/commands/FIX.md`
2. Проверить список проблем с командами
3. Исправить проблемы по порядку

---

### Шаг 3: Протестировать каждую команду

**Чек-лист для каждой команды:**
- [ ] Linux: работает
- [ ] Фразы RU: есть
- [ ] Звуки: есть
- [ ] Файл: синтаксис валиден

---

### Шаг 4: Обновить документацию

**После проверки:**
- [ ] Отметить в `resources/commands/ADD.md` какие команды проверены
- [ ] Отметить в `resources/commands/FIX.md` какие проблемы исправлены
- [ ] Обновить `EVERY_CHECK.md` если изменилась проверка
