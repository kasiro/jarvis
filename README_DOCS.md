## 🚀 Быстрый старт

### 1. Прочитать главное:
```bash
cat QWEN.md
cat EVERY_CHECK.md
```

### 2. Проверить проект:
```bash
git status
cargo check --workspace
```

### 3. Собрать:
```bash
./rebuild.sh --fast
```

### 4. Запустить:
```bash
./jarvis.sh
```
